# JSON 数据格式

## JSON 数据格式

JSON 支持三种类型的数据格式：简单值、对象和数组。

## 简单值

最有效的简单值可以是：字符串、数字、布尔值和 null。

::: warning

- JSON 不支持 JavaScript 特有的数据类型，如：undefined、函数和正则表达式。
- JSON 中的字符串必须使用双引号，而不能使用单引号。

:::

## 对象

JSON 中的对象是一种**无序的**键/值对集合。

对象的键必须是字符串，值可以是简单值、对象或数组。

## 数组

JSON 中的数组是一种**有序的**值的列表。

数组的值可以是简单值、对象或数组。
