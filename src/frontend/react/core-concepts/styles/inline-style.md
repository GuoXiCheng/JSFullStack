# 内联样式

## 什么是内联样式

内联样式是一种将 CSS 样式直接写在 JSX 元素上的方式。在 React 中，可以使用内联样式来设置元素的样式。

## 基本用法

使用`style`属性设置样式，样式的属性名使用驼峰命名法。
