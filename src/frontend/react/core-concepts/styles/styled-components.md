# Styled Components

## 什么是 Styled Components

Styled Components 是一个 CSS-in-JS 库，它可以使用 JavaScript 来编写 CSS 样式。

Styled Components 通过创建 React 组件的方式来定义样式，这样可以在组件内部定义样式，而不是在全局定义样式。

## 安装

```bash
npm install styled-components
```
