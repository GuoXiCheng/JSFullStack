# Redux Toolkit

## 什么是 Redux Toolkit

Redux Toolkit 是一个官方的 Redux 标准工具包，它可以帮助你编写 Redux 逻辑更少、更简单。它包含了一些有用的函数和工具，可以帮助你编写 Redux 代码更少，更简洁。

## 安装

```bash
npm install @reduxjs/toolkit react-redux
```
